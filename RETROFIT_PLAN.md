## How to Retrofit Each Chapter

For each Tier B/C chapter, do this **mini-pass** (~1-2 hours):

1. **Read the chapter end-to-end.** Where does it leap from hook to mystery?
2. **Identify or add a CONCEPT slide** before the first mystery.
3. **Audit each mystery:** does the Concept solve it? If not, reorder or add scaffolding.
4. **Add 2-3 quiz questions** hitting:
   - 1 graph/data interpretation
   - 1 toddler-explain
   - 1 troubleshooting / what-if
5. **Add 1-2 homework items** if missing CER format or Toddler-explain.
6. **Audit distractors:** rewrite any "Magic" / "No." options as plausible misconceptions.
7. **Add 1 spiral callback** to a prior chapter (where applicable).
8. **Validate:** JS check, slide count, template-leak checks.

For Tier D chapters (currently only B-12), do a **larger pass** (~half-day):

1. Plan Concept anchor + restructure mysteries to flow CEA.
2. Add full set of 7 quality patterns.
3. Rewrite 50%+ of quiz with thought-provoking distractors and hints.
