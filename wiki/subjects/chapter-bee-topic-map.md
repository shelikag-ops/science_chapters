## Coverage summary (rough)

For the four target bees, expected coverage assuming all 38 chapters reach Grade A quality and have bee mode enabled:

| Bee | Estimated chapter-only coverage | Gap to fill from external resources |
|---|---|---|
| **SOF ISO Class 3 — Science section** | ~85% | Logical Reasoning section (use MTG workbook reasoning chapters) |
| **Silverzone iOS Class 3** | ~85% | Same gap as SOF |
| **NSTSE Class 3** | ~85% | Same gap; plus NCERT-specific phrasing — use Unified Council sample papers |
| **NSB Elementary (2nd-3rd grade)** | ~70% | Math (use AoPS Beast Academy 2/3), more astronomy depth (NASA Space Place), engineering vocabulary |
| **NSF JSC (Grades 1-3)** | ~95% | Almost fully covered by chapters + DK encyclopedia |

The big takeaway: **the BFSU chapter set is already a near-complete prep curriculum** for the India olympiads and NSF JSC. The US National Science Bee needs supplemental math + astronomy.
